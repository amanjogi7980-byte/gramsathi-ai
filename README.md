# GramSathi AI 🌾🤖

## AI Powered Rural Assistance Platform

GramSathi AI is an intelligent rural assistance platform designed to bridge the gap between citizens and digital government services. The platform helps users discover government schemes, access document guidance, submit complaints, track complaint status, and interact with an AI-powered assistant using both text and voice.

---

## 🚀 Features

### 🤖 AI Assistant

* AI-powered chatbot
* Voice Input (Speech Recognition)
* Voice Output (Text-to-Speech)
* Hindi and English support
* Rural service guidance

* Category-wise scheme search
* Farmer Schemes

### 📄 Document Checker

## 🎯 Problem Statement

Many rural citizens face challenges while accessing government schemes, obtaining document-related information, and submitting complaints through digital platforms.

GramSathi AI provides a simple and user-friendly solution that enables citizens to access multiple government services from a single platform.

## ⚙ Installation Guide

### 1. Clone Repository

```bash
git clone https://github.com/your-username/gramsathi-ai.git
```

### 2. Open Project Folder

```bash
cd gramsathi-ai
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run Backend Server

```bash
cd backend
python app.py
```

Backend will run at:

```text
http://127.0.0.1:5000
```

### 5. Run Frontend

Open:

```text
frontend/index.html
```

using Live Server in VS Code.
